Code developed as a part of Advanced Robotic Systems Course (QMUL-ECS7004P)

To run the package:

1) Unzip "AR_week4_test.zip" as "AR_week4_test" folder in your catkin workspace.
2) Build the catkin workspace.
3) Run the 'cubic_traj_gen.launch' launch file.

* Build and running process is repeated in the screencast provided
  within the zip file. *

** Time update factor in "plot_cubic_traj.py" might be changed if the plots
   are too course or too fine to the liking. **

Code developed as a part of Advanced Robotic Systems Course (QMUL-ECS7004P)

Note that this code can be run in a ROS environment. The screencast provided with this project uses ROS Melodic and Ubuntu 18.04.

To run the package:

1) Unzip "AR_week4_test.zip" as "AR_week4_test" folder in your catkin workspace.
2) Build the catkin workspace.
3) Run the 'cubic_traj_gen.launch' launch file.

A sample execution can be found from [this](https://drive.google.com/file/d/1ilSB_frgNKHjQqqKw-OeBiYaihFpxrm0/view?usp=sharing) link.
